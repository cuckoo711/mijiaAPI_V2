## 运行测试

### 运行所有测试

```bash
source .venv/bin/activate
python -m pytest tests/ -v
```

### 生成覆盖率报告

```bash
source .venv/bin/activate
python -m pytest tests/ --cov=mijiaAPI_V2 --cov-report=html --cov-report=term
```

### 查看 HTML 覆盖率报告

```bash
open htmlcov/index.html
```

### 运行特定模块测试

```bash
# 测试 HTTP 客户端
```
